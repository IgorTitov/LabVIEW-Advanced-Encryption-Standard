Installation

This section documents installation instructions for AES LabVIEW Library.

  1. Download VIPM package of AES Encryption Library
  2. Open it with VI Package Manager (available for free at www.jki.net/vipm) and follow instructions
  3. Restart LabVIEW

System Use RAM Processor
Development 1 GB 1 GHz or better
Deployment 256 MB 600 MHz or better

Software Support in AES LabVIEW Library

Operating System Support

The following operating systems are supported in AES LabVIEW Library 1.0.0:

  • Windows
  • Mac OS
  • Linux

Note  This library was written in pure LabVIEW, therefore it's compatible with any operating system that LabVIEW itself can run on.

Caution  AES 1.0.0 uses floating point operations and for this reason currently can't be used on FPGA targets. For FPGA you can use AES library supplied by National Instruments.
